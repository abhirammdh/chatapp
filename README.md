# chatapp# User-to-User Real-Time Chat App

Private messaging with Socket.IO + Streamlit UI (localStorage for usernames).

## Quick Start
1. Backend: `cd backend && npm install && npm start`
2. Frontend: `cd frontend && pip install -r requirements.txt && streamlit run app.py`
3. Open http://localhost:8501 â†’ Test in multiple tabs.

## Features
- Real-time private chats
- Online users list (click to chat)
- Username saved locally (remembers + editable)

## Deploy
- Backend: Render.com (Web Service)
- Frontend: share.streamlit.io (update JS socket URL to live backend)
